# Images pour le slider Avant/Après

## Fichiers à ajouter dans ce dossier :

### 1. `before-example.jpg`
- **Description** : Photo "AVANT" - prise au smartphone
- **Recommandations** :
  - Photo d'un ouvrage tricot/crochet prise avec un smartphone
  - Éclairage ordinaire, fond quelconque
  - Format recommandé : 1920x1080 ou 16:9
  - Poids max : 500 Ko

### 2. `after-example.jpg`
- **Description** : Photo "APRÈS" - retouchée par YarnFlow IA
- **Recommandations** :
  - La MÊME photo retouchée avec YarnFlow AI Studio
  - Doit montrer la différence (meilleur éclairage, fond propre, etc.)
  - Format recommandé : 1920x1080 ou 16:9
  - Poids max : 500 Ko

## Important
Les deux images doivent être **exactement le même ouvrage** pour que la comparaison soit percutante !

## Exemple d'utilisation
Une fois les images ajoutées, le slider permettra de glisser entre les deux versions et voir la transformation en temps réel.
